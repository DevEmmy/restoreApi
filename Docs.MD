# API Docs

To sign up
- POST: "/users/signup"
### Parameters
- email
- password

To sign in
- POST: "/users/signin"
### Parameters
- email
- password

To upload a document
- POST: "/documents/upload"
### Parameters
- document
- title

To update profile
- POST: "/profile/update-profile"
### Parameters
- firstName
- lastName
- matricNumber
- level
- college
- department
- telephone
